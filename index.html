2025

Automated Raman Analyzer Virtualized Test UI

Role: Engineering, tooling, CI orchestration

Designed and implemented a Python-based desktop application for real-time visualization and analysis of Raman spectroscopy data from a Kaiser Raman analyzer via OPC UA. The system features a structured startup workflow that generates client certificates, configures network connections, and allows operation in either live acquisition mode or offline model-testing mode. The primary interface provides dynamic, multi-channel spectral plotting using Matplotlib, with keyboard-driven navigation, flexible channel layouts, and continuous updates as new data is acquired. Integrated a modular prediction layer that loads user-supplied analytical models to perform per-channel inference on incoming spectra and present predictive results alongside live measurements.
